Mac 搭建属于自己的hexo(小白鼠级别)

一、Mac下如何使用hexo搭建自己的blog

最近有些兴趣想搞自己的blog,突然听同学说hexo好用（主要是可以装逼）于是就想搭建自己的blog,作者属于小白鼠的级别所以就开始在某度开始搜教程，结果。。。。。。。。跳坑开始了。。。。。。鉴于此种情况怀着拯救世界的心里开始了自己的第一篇博客

话不多说开始上干货：

先屡一下思路

1.首先hexo是基于nodejs的，所以必须安装nodejs

2.安装nodejs方法很多，我选择homebrew安装方式，所以需要安装它(假如你看了某博客使用了dmg/pkg的方式安装了node.js,那么我只能为你祈祷)

3.安装homebrew就很简单了，mac自带ruby脚本功能，一句话搞定

4.hexo提交部署github需要使用git工具，所以需要安装git，用homebrew的话也是一句话搞定

5.OK整理一下安装顺序（homebrew-nodejs-hexo-git）

1.0 安装brewhome，一句话搞定

2.0 安装node.js，brewhome安装，一句话搞定(brew install node)

3.0 安装hexo,用nodejs自带npm安装

4.0 安装git(未使用xcode的同学),homebrew安装，一句话搞定

4.1.0 配置

4.1.1 检查SSH key

4.1.2 备份已有的key，（如果有的话）

4.1.3 生成SSH key

4.1.4 将SSH key添加到Github

4.1.5 测试连接

4.1.6 设置个人信息

5.0 修改hexo根目录下_config.yml文件（xxx为你的github账户名称）

6.0 注册github账号，新建名为xxx.github.io的repository

7.0 到这，Hexo博客搭建已经完成了，并且可以git提交到github上，通过访问xxx.github.io就可以访问本博客(每次写完自己blog后)
